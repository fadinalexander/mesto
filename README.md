# Проект: Место

### Описание

Проект является одностраничным сайтом с реализацией элементов JavaScript (работа с формами) и адаптивной верстки для экранов от 320px.

### Технологии

В проекте применяются технологии:

* JavaScript
* Validation
* Flex
* Grid
* @media request
* файловая структура по БЭМ (Nested)
